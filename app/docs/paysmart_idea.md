# MVP Solution

A simple group-based expense + balance system with instant settlement

Core flow:

User tops up (already built ✅)
Creates a group (e.g. “Flat”, “Trip”, “Team”)
Adds expenses:
“£42 groceries”
split automatically
App shows:
“You owe £18 to Alex”
One-tap settle:
Pay directly inside PaySmart
🧠 Why This Works

Because it combines:

💸 money storage (you already have)
🧾 context (expenses)
🔁 repeat usage

This creates a loop, not a one-off action.
